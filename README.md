# Interview Question Generator

An AI-powered web application that generates interview questions based on user skills or resumes and evaluates answers using NLP.

## Features
- Resume-based question generation
- AI answer evaluation
- Performance dashboard
- Voice input support
- PDF report generation

## Tech Stack
- Python (Flask)
- Scikit-learn (TF-IDF, Cosine Similarity)
- HTML, CSS, JavaScript
- PyPDF2, ReportLab

## Run Locally
```bash
pip install -r requirements.txt
python app.py
